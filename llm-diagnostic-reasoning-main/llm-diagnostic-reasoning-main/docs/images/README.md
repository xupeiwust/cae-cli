# 可视化示例图片

## 📸 如何添加示例图片

1. 运行诊断生成可视化：
```bash
python diagnose.py --demo hvac --output-viz reasoning_chain.png
```

2. 将生成的图片复制到这个目录：
```bash
cp reasoning_chain.png docs/images/
cp bayesian_network.png docs/images/
```

3. 或者使用截图工具捕获运行效果
